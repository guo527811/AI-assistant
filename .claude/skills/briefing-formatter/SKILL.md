# 简报格式化工具

## 用途
生成结构化的 AI 新闻简报和每日学习卡片。

## 何时使用
- 生成新闻简报、日报、周报
- 创建结构化的学习笔记
- 任何需要"标题 -> 摘要 -> 分类 -> 建议"格式的输出

## 操作步骤

### 1. 确定日期
- 获取当天日期，格式：YYYY年MM月DD日（星期X）
- 文件名：YYYY-MM-DD

### 2. 新闻简报格式

```markdown
# [类型] -- YYYY年MM月DD日（星期X）

> 共收录 X 条 | 标签：分类标签

## 今日要闻 TOP 3

### 1. [标题]
- **来源：** [来源名称](链接)
- **标签：** 分类
- **摘要：** 2-3句中文概括
- **为什么重要：** 一句话

### 2. ...
### 3. ...

## 分类内容

### 分类标签
- **[标题]**：[摘要]（[来源](链接)）

## 建议
1. 具体建议
2. 具体建议
```

### 3. 学习卡片格式

```markdown
# [主题] -- YYYY年MM月DD日（星期X）

> 今日主题：[核心关键词] | 共 X 个学习点

## 必学内容

### 知识点 N：[标题]
- **来源：** [链接]
- **难度：** 入门 / 进阶 / 高级
- **求职相关度：** 1-5 星
- **核心概念：** 3-5 句话说清原理
- **代码示例：** 贴关键代码，带注释
- **实操练习：** 一个可执行的小任务

## 扩展阅读

- **[标题]**：[1-2句简介]（[链接]）

## 实用技巧

1. **技巧名：** 一句话说明

## 今日总结

[200 字以内的个人理解]
```

### 4. 标签体系

**新闻标签：**
- 新技能 -- 新技术、新功能发布
- 新工具 -- 新工具、新框架
- 行业动态 -- 市场趋势、企业动向
- 政策/投融资 -- 政策法规、融资信息

**学习标签：**
- 面试常考 -- 面试高频考点
- 实战必会 -- 工作中一定用到
- 了解即可 -- 知道就行，不用深究

### 5. 质量要求
- 所有内容必须来自真实来源，附链接
- 禁止编造任何信息
- 每条摘要控制在 150 字以内
- 优先选取与 Claude、AI Agent 相关的内容
- 代码示例必须完整可运行
- 文件名统一用 YYYY-MM-DD 格式

### 6. 输出路径规范

| 内容类型 | 输出路径 |
|----------|----------|
| 新闻简报 | `E:/AI助手/新闻简报/YYYY-MM-DD/今日AI新闻.md` |
| 每日学习 | `E:/AI助手/学习资料/每日学习/YYYY-MM-DD-Claude-Code学习.md` |
| 周报 | `E:/AI助手/学习资料/学习周报-第N周.md` |
| 岗位汇总 | `E:/AI助手/求职/岗位追踪/YYYY-MM-DD-岗位汇总.md` |
